\doxysubsection{Detailed Description}
This file contains the example of how to do with \mbox{\hyperlink{classMachineState}{Machine\+State}} class. 

The basic usage of these \mbox{\hyperlink{classMachineState}{Machine\+State}}, you must have 3 kind of things on each state, there are input, outpu, and action

The input is the condition that what the trigger for another state going into current state

The output is the condition that what the trigger for current state going into another state

The action is the action that you want to do with your state
